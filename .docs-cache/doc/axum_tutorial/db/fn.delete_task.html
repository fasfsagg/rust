Expand description

从数据库中删除任务 (Delete Operation)

【功能】: 根据 ID 查找并移除任务。

§【参数】

  • db: &Db - 数据库实例的引用。
  • id: Uuid - 要删除的任务的 UUID。[[所有权: 拷贝]]

§【返回值】

  • -> Result<Task>:
    • Ok(Task): 如果删除成功，返回被删除的那个任务实例的【所有权】。
    • Err(AppError::TaskNotFound): 如果数据库中不存在具有该 ID 的任务。
